# 5-Day AI Agents: Intensive Vibe Coding Course With Google

# Day-1: Introduction to Agents & Vibe Coding

Based on the Day 1 summary podcast and the Kaggle whitepaper, "The New SDLC with Vibe Coding," we explored how AI-native tools are redefining software development. The traditional Software Development Life Cycle (SDLC) is shifting from manual coding toward a system orchestration paradigm. Developers now use conversational agents to translate natural language intent into functional systems, turning development into a rapid loop of real-time ideation, agent execution, and intent validation.

The tasks assigned on Day 1 included completing two hands-on codelabs: "Get started with Antigravity 2.0 and IDE" and "Build a Web Application and Deploy." These codelabs introduced the agentic development workflow, demonstrating how to collaborate with AI coding assistants in modern IDE workspaces and leverage automated tools to build, verify, and host static web applications with speed and precision.

## Project: QR Code Generator

This project was entirely vibe coded, leveraging AI agents to design and implement a clean, single-page QR Code Generator using vanilla HTML, CSS, and JavaScript. Powered by the QRCode.js CDN, the application instantly generates QR codes for both URLs and plain text on button click or pressing Enter, clears previous results to prevent visual clutter, and exposes a PNG download option after rendering. It runs completely client-side, making it highly optimized for fast, serverless hosting on static platforms.

**Live Website:** https://chandan-ng.github.io/QR-Code-Generator/
